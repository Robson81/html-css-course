Recuperando Dados de Formulário

Preencha o formulário e veja como o JavaScript recupera os dados

Dados Recuperados:

Como funciona:

Quando você clica no botão "Enviar Dados", o JavaScript:

  1. Impede o envio padrão do formulário (que recarregaria a página)
  2. Recupera os valores dos campos usando getElementById()
  3. Exibe os dados recuperados na área de resultados

O código JavaScript utilizado:

const form = document.getElementById('loginForm');

form.addEventListener('submit', function(event) {
  event.preventDefault(); // Impede o envio padrão do formulário

  // Recupera os valores dos campos
  const email = document.getElementById('email').value;
  const senha = document.getElementById('senha').value;

  // Exibe os resultados
  document.getElementById('emailResult').textContent = 'E-mail: ' + email;
  document.getElementById('senhaResult').textContent = 'Senha: ' + senha;
  document.getElementById('result').style.display = 'block';
});
