CSS utilizado para corrigir:
#nav-target {
    position: absolute;
    top: 100%;
    right: 0;
    left: auto;
    width: 200px;
    background-color: #343a40;
    z-index: 1000;
    border-radius: 0 0 5px 5px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}

Como funciona a solução

O problema original acontecia porque o menu dropdown do Bootstrap é projetado para aparecer abaixo de toda a barra de navegação. Para corrigir isso:

  1. Adicionamos position: absolute ao menu dropdown
  2. Posicionamos o menu no canto superior direito com top: 100%; right: 0;
  3. Definimos uma largura fixa para o menu com width: 200px;
  4. Garantimos que o menu fique acima de outros elementos com z-index: 1000;
  5. Adicionamos um sombreamento para melhor visualização

Redimensione a janela para ver o menu responsivo em ação. Em telas menores, o ícone hamburger aparecerá e, ao clicar, o menu será exibido abaixo dele.
